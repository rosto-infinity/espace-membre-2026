# Qu'est-ce que PHP ?

Le PHP (acronyme récursif pour **PHP: Hypertext Preprocessor**) est un langage de script open-source extrêmement utilisé. Il est particulièrement adapté au développement web et peut être directement intégré (embedded) dans le code HTML.

## Pourquoi apprendre le PHP ?

PHP propulse environ **77 % de tous les sites web** dont le langage de programmation côté serveur est connu, incluant des géants comme WordPress, Facebook et Wikipédia. Voici pourquoi il reste incontournable :

- **Puissance Côté Serveur** : PHP s'exécute sur le serveur et génère du HTML qui est ensuite envoyé au navigateur du client.
- **Intégration de Bases de Données** : Il se connecte nativement et facilement à MySQL, PostgreSQL, SQLite et bien d'autres.
- **Multiplateforme** : Il fonctionne sur Windows, Linux, macOS et la plupart des serveurs web (Apache, Nginx).
- **Écosystème Vaste** : Des milliers de frameworks (Laravel, Symfony) et de plateformes CMS (WordPress, Drupal) l'utilisent.
- **Développement Actif** : PHP 8.x apporte des fonctionnalités modernes comme la compilation JIT, les attributs et les types d'union.

## Comment fonctionne le PHP

```text
1. L'utilisateur demande une page .php (via son navigateur)
2. Le serveur web (Apache/Nginx) transmet la requête à l'interpréteur PHP
3. PHP exécute le code, interrogeant éventuellement une base de données
4. PHP génère un flux de sortie HTML
5. Le serveur renvoie ce HTML au navigateur de l'utilisateur
```

## PHP face aux autres langages

| Caractéristique         | PHP      | Node.js            | Python             |
| ----------------------- | -------- | ------------------ | ------------------ |
| Courbe d'apprentissage  | Facile   | Modérée            | Facile             |
| Focus Web               | Natif    | Basé sur framework | Basé sur framework |
| Hébergement             | Partout  | Spécialisé         | Spécialisé         |
| Gestionnaire de paquets | Composer | npm                | pip                |

L'intégration étroite de PHP avec les serveurs web et le HTML en fait un excellent choix comme premier langage de programmation côté serveur à apprendre.
